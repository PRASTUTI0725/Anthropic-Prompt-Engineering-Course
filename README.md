# Anthropic Prompt Engineering Interactive Tutorial Notes

This repository contains my detailed notes and exercises from **Anthropic's Prompt Engineering Interactive Tutorial**, covering all 9 chapters and the appendix.

## About the Course

This course is designed to provide a step-by-step understanding of how to engineer optimal prompts for Claude. After completing this course, I have learned to:

- Master the basic structure of effective prompts
- Recognize common failure modes and address them efficiently
- Understand Claude's strengths and weaknesses
- Build complex prompts for real-world use cases

## Course Structure

The course is divided into **9 chapters** with accompanying exercises, plus an appendix for advanced topics:

### Beginner
- **Chapter 1:** Basic Prompt Structure
- **Chapter 2:** Being Clear and Direct
- **Chapter 3:** Assigning Roles

### Intermediate
- **Chapter 4:** Separating Data from Instructions
- **Chapter 5:** Formatting Output & Speaking for Claude
- **Chapter 6:** Precognition (Step-by-Step Thinking)
- **Chapter 7:** Using Examples

### Advanced
- **Chapter 8:** Avoiding Hallucinations
- **Chapter 9:** Building Complex Prompts (Industry Use Cases)
  - Chatbot
  - Legal Services
  - Financial Services (Exercise)
  - Coding (Exercise)

### Appendix
- Chaining Prompts
- Tool Use
- Search & Retrieval

## Repository Contents
- Markdown notes for each chapter and exercise
- Examples and explanations of prompt engineering techniques
